# Reflection – Person 2

## Frontend Logic, Routing and State Management

Working on the EasyBuy project as Person 2 helped me gain practical experience in frontend development, routing, state management, and frontend-backend integration. My main responsibility was handling the frontend logic of the application and ensuring smooth communication between the frontend and backend systems.

At the beginning of the project, I had limited experience with frontend routing and authentication flow. Through this project, I learned how Next.js App Router works and how different pages can be connected using navigation and dynamic filtering. Implementing protected routes and category-based navigation helped me better understand frontend application flow.

One of the main tasks I completed was implementing authentication functionality. I created login and registration pages and connected them with backend APIs using Fetch API. During development, I faced issues such as server connection failures, incorrect authentication responses, and routing problems. Solving these issues improved my debugging and problem-solving skills.

I also learned how to manage authentication state using Zustand. Before this project, I did not fully understand how global state management works in frontend applications. By implementing login state handling and protected navigation, I gained a better understanding of state management concepts.

Another important feature I implemented was category navigation and product filtering. Initially, filtering did not work correctly because category names were inconsistent across components and product data. After fixing the category structure, the filtering system worked properly and improved the user experience.

I also improved the user interface using Tailwind CSS. I adjusted placeholder visibility, text colors, navbar hover effects, and layout responsiveness to make the application more user-friendly and visually clear. This helped me understand the importance of UI design and usability in web applications.

Through this project, I improved my skills in:
- Next.js routing
- frontend logic implementation
- API integration
- authentication flow
- Zustand state management
- responsive UI development
- debugging and error handling
- teamwork and frontend-backend coordination

Overall, this project gave me valuable hands-on experience in frontend web development and helped me understand how frontend systems interact with backend services in a real-world application.
